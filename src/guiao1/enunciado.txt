Guião 1
O teu programa deve:

Ler uma linha
Imprimir o conteúdo da stack (sem espaços)
Exemplo
Input
17 11 - ( 18 9 /
Output
52
Input
17 11 - ( 18 9 / *
Output
10
Input
5 3 ^ 12 &
Output
4
Relembro que a submissão deve seguir o seguinte protocolo:

Deve ter a extensão zip e ser um zip
Deve conter uma pasta chamada code
A pasta code deve conter todo o código necessário para compilar o programa
A pasta só deve ter os ficheiros com a extensão C necessários para compilar o programa
O código dessa pasta será compilado da seguinte forma: gcc -std=gnu11 -Wall -Wextra -pedantic-errors -O main.c -lm
