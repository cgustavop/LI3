Guião 3
Todas as condições do guião 2 se aplicam aqui também: o teu programa recebe várias linhas. A primeira linha é o programa e as próximas linhas podem ser lidas com o comando l.

Todos os testes que usarem o comando l irão sempre seguir com uma conversão para inteiro ou float.

Exemplo 1
Este programa escolhe o maior dos dois valores que estão no topo da stack

Input
7 14.5 14 e>
Output
714.5
Exemplo 2
Como ambos os valores são verdadeiros (porque são diferentes de zero) o & com shortcut escolhe o primeiro.

Input
4.9 4 e|
Output
4.9
Exemplo 3
Este programa lê dois números e imprime o maior.

Input
l i :A l i :B > A B ?
7
4
Output
7
Exemplo 4
As condições colocam sempre um long na stack.

Input
48 c ! 0.0 ! 2 ! 0 c !

Output
0101
